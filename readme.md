# Balance électronique pour mesure du Centre de Gravité des modèles réduits d'avion.

*** Attention ! En cours de développement, non terminé pour utilisation ! ***

Balance basée sur :
- Une carte microcontrôleur WiFi ESP8266 de type Wemos D1 mini, 
- deux capteurs de masse formés par une cellule de charge et d'un module amplificateur HX711,
- un écran I2C SSD1306 de 0.96 pouce (résolution 128x64).

La balance affiche sur son écran les masse mesurées sur l'appuis bord d'attaque, sur l'appuis bord de fuite, la masse totale et la position du centre de gravité en millimètres mesurée à partir du bord d'attaque de l'aile.
